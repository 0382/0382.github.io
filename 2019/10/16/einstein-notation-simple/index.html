关于三维矢量和三维二阶张量的一些运算

发表于2019-10-16更新于2020-10-30阅读次数

使用爱因斯坦求和约定以及 Kronecker delta 记号来推导类似的公式会比较方便.

以下所有的矢量或者张量都是三维的

矢量符号或者张量符号就不打了，大写字母表示二阶张量，小写字母表示矢量，希腊字母表示标量

矢量的点乘、叉乘和并矢

  • 点乘

    ϕ=aibi\phi = a_ib_i ϕ=ai​bi​

  • 叉乘

    ci=ϵijkajbkc_i = \epsilon_{ijk}a_jb_k ci​=ϵijk​aj​bk​

    其中，ϵijk\epsilon_{ijk}ϵijk​为三阶反对称张量.

  • 并矢

    Tij=aibjT_{ij} = a_ib_j Tij​=ai​bj​

二阶张量和矢量运算

  • 张量和矢量点乘

    bi=Aijajb_i = A_{ij}a_j bi​=Aij​aj​

  • 矢量和张量点乘

    ci=ajAjic_i = a_jA_{ji} ci​=aj​Aji​

  • 矢量和张量叉乘

    Tij=ϵiklakAljT_{ij} = \epsilon_{ikl}a_{k}A_{lj} Tij​=ϵikl​ak​Alj​

二阶矢量的特殊运算

  • 点乘（虽然不算什么特殊运算）

    Cij=AikBkjC_{ij} = A_{ik}B_{kj} Cij​=Aik​Bkj​

  • 双点乘

    ϕ=A:B=AijBji=trace(AB)\phi = A:B = A_{ij}B_{ji} = trace(AB) ϕ=A:B=Aij​Bji​=trace(AB)

  • 迹

    trace(A)=Aii=Aijδji=A:Itrace(A) = A_{ii} = A_{ij}\delta_{ji} = A:I trace(A)=Aii​=Aij​δji​=A:I

  • 矩阵的行列式（可以看成三个矢量加上反对称张量的缩并，然而并没有什么卵用）

det(A)=ϵijkA1iA2jA3k=ϵijkAi1Aj2Ak3det(A) = \epsilon_{ijk}A_{1i}A_{2j}A_{3k} = \epsilon_{ijk}A_{i1}A_{j2}A_{k3} det(A)=ϵijk​A1i​A2j​A3k​=ϵijk​Ai1​Aj2​Ak3​

梯度，散度，旋度，nabla 算子与矢量的并矢

  • ∇\nabla∇算子

    ∇=∂∂xex⃗+∂∂yey⃗+∂∂zez⃗\nabla = \dfrac{\partial}{\partial x}\vec{e_x} + \dfrac{\partial}{\partial y}\vec{e_y} + \dfrac{\partial}{\partial z} \vec{e_z} ∇=∂x∂​ex​​+∂y∂​ey​​+∂z∂​ez​​

    这是可以看成一个矢量的，也可以写成

    ∇i=∂i=∂∂xi\nabla_i = \partial_i = \dfrac{\partial}{\partial x_i} ∇i​=∂i​=∂xi​∂​

  • 矢量的散度

ϕ=∇⋅a⃗=∂iai\phi = \nabla\cdot\vec{a} = \partial_i a_i ϕ=∇⋅a=∂i​ai​

  • 标量的梯度

    ai=∂iϕa_i = \partial_i \phi ai​=∂i​ϕ

  • ∇\nabla∇算子与矢量的并矢其实就矢量的梯度，形成二阶张量

    Tij=∇a⃗=∂iajT_{ij} = \nabla\vec{a} = \partial_i a_j Tij​=∇a=∂i​aj​

  • 矢量的旋度

    bi=∇×a⃗=ϵijk∂jakb_i = \nabla\times\vec{a} = \epsilon_{ijk}\partial_ja_k bi​=∇×a=ϵijk​∂j​ak​

  • 并矢的散度

    ∇⋅(a⃗b⃗)=∂i(aibj)=(∂iai)bj+ai(∂ibj)=(∇⋅a⃗)b⃗+a⃗⋅∇b⃗\nabla\cdot(\vec{a}\vec{b}) = \partial_i(a_ib_j) = (\partial_ia_i)b_j + a_i(\partial_ib_j) = (\nabla\cdot\vec{a})\vec{b} + \vec{a}\cdot\nabla\vec{b} ∇⋅(ab)=∂i​(ai​bj​)=(∂i​ai​)bj​+ai​(∂i​bj​)=(∇⋅a)b+a⋅∇b

  • 并矢的旋度

    ∇×(a⃗b⃗)=ϵijk∂j(akbl)=ϵijk(∂jak)bl+ϵijkak(∂kbl)=(∇×a⃗)b⃗+a⃗×(∇b⃗)\nabla\times(\vec{a}\vec{b}) = \epsilon_{ijk}\partial_j(a_kb_l) = \epsilon_{ijk}(\partial_ja_k)b_l+\epsilon_{ijk}a_k(\partial_kb_l) = (\nabla\times\vec{a})\vec{b} + \vec{a}\times(\nabla\vec{b}) ∇×(ab)=ϵijk​∂j​(ak​bl​)=ϵijk​(∂j​ak​)bl​+ϵijk​ak​(∂k​bl​)=(∇×a)b+a×(∇b)

一些公式

  • ∇\nabla∇算子的二阶作用

∇2ϕ=(∇⋅∇)ϕ∇×(∇×a⃗)=∇(∇⋅a⃗)−∇2a⃗∇×∇ϕ=0∇⋅(∇×a⃗)=0\begin{aligned} \nabla^2\phi &= (\nabla\cdot\nabla)\phi \\ \nabla\times(\nabla\times\vec{a}) &= \nabla(\nabla\cdot\vec{a}) - \nabla^2\vec{a} \\ \nabla\times\nabla\phi &= 0 \\ \nabla\cdot(\nabla\times\vec{a}) &= 0 \end{aligned} ∇2ϕ∇×(∇×a)∇×∇ϕ∇⋅(∇×a)​=(∇⋅∇)ϕ=∇(∇⋅a)−∇2a=0=0​

以其中第二个式子的证明为例

∇×(∇×a⃗)=ϵijk∂j(ϵklm∂lam)=ϵijkϵklm∂j∂lam=(δilδjm−δimδjl)∂j∂lam=∂i(∂jaj)−∂j∂jai=∇(∇⋅a⃗)−∇2a⃗\begin{aligned} \nabla\times(\nabla\times\vec{a}) &= \epsilon_{ijk}\partial_j(\epsilon_{klm}\partial_la_m) \\ &= \epsilon_{ijk}\epsilon_{klm}\partial_j\partial_la_m \\ &= (\delta_{il}\delta_{jm}-\delta_{im}\delta_{jl})\partial_j\partial_la_m \\ &= \partial_i(\partial_ja_j) - \partial_j\partial_ja_i \\ &= \nabla(\nabla\cdot\vec{a})-\nabla^2\vec{a} \end{aligned} ∇×(∇×a)​=ϵijk​∂j​(ϵklm​∂l​am​)=ϵijk​ϵklm​∂j​∂l​am​=(δil​δjm​−δim​δjl​)∂j​∂l​am​=∂i​(∂j​aj​)−∂j​∂j​ai​=∇(∇⋅a)−∇2a​

Kronecker delta
