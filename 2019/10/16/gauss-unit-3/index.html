高斯单位制（其三）

发表于2019-10-16更新于2020-10-30阅读次数

这一节开始推导一下高斯单位制下，静电场和静磁场的相关公式。

静电场

静电场的出发点自然式麦克斯韦方程组关于电场的两个，因为是静电场，所以磁场变化项就为零了：

∇⋅D⃗=4πρf∇×E⃗=0D⃗=ϵE⃗\begin{aligned} \nabla\cdot\vec{D} &= 4\pi\rho_f \\ \nabla \times \vec{E} &= 0 \\ \vec{D}& = \epsilon\vec{E} \end{aligned} ∇⋅D∇×ED​=4πρf​=0=ϵE​

根据∇×E⃗=0\nabla\times \vec{E} = 0∇×E=0，静电场下可以定义电势

ψ(r⃗2)−ψ(r⃗1)=−∫r⃗1r⃗2E⃗⋅dl⃗\psi(\vec{r}_2) - \psi(\vec{r}_1) = -\int_{\vec{r}_1}^{\vec{r}_2}\vec{E}\cdot d\vec{l} ψ(r2​)−ψ(r1​)=−∫r1​r2​​E⋅dl

自然，很容易得到

E⃗=−∇ψ\vec{E} = -\nabla \psi E=−∇ψ

通过点电荷电势，可以推出一般的电势公式

ψ(r⃗)=∫ρ(r⃗′)dV′∣r⃗−r⃗′∣\psi(\vec{r}) = \int\dfrac{\rho(\vec{r}')dV'}{|\vec{r}-\vec{r}'|} ψ(r)=∫∣r−r′∣ρ(r′)dV′​

再根据∇⋅D⃗=4πρf,D⃗=ϵE⃗\nabla\cdot\vec{D}=4\pi\rho_f,\vec{D}=\epsilon\vec{E}∇⋅D=4πρf​,D=ϵE，于是有静电场的泊松方程

∇2ψ=−4πϵρf\nabla^2\psi = -\dfrac{4\pi}{\epsilon}\rho_f ∇2ψ=−ϵ4π​ρf​

电偶极子

定义和SI制一样，p⃗=q(r⃗+−r⃗−)\vec{p} = q(\vec{r}_+-\vec{r}_-)p​=q(r+​−r−​)或者一般式

p⃗=∫ρr⃗dV\vec{p} = \int \rho \vec{r} dV p​=∫ρrdV

远场的偶极子电势

ψ=p⃗⋅r⃗r3\psi = \dfrac{\vec{p}\cdot\vec{r}}{r^3} ψ=r3p​⋅r​

偶极子受力为，

F⃗=−∇U=∇(p⃗⋅E⃗外)=p⃗⋅∇E⃗外\vec{F} = -\nabla U = \nabla (\vec{p}\cdot \vec{E}_{\text{外}}) = \vec{p}\cdot\nabla\vec{E}_{\text{外}} F=−∇U=∇(p​⋅E外​)=p​⋅∇E外​

受到的力矩为

N⃗=p⃗×E⃗\vec{N} = \vec{p}\times\vec{E} N=p​×E

这些都和SI制差不太多，至于电四极矩之类的，应该也很好推导。

静磁场

静磁场出发点也是麦克斯韦方程组

∇⋅B⃗=0∇×H⃗=4πcj⃗fB⃗=μH⃗\begin{aligned} \nabla \cdot \vec{B} &= 0 \\ \nabla \times \vec{H} &= \dfrac{4\pi}{c} \vec{j}_f \\ \vec{B} &= \mu\vec{H} \end{aligned} ∇⋅B∇×HB​=0=c4π​j​f​=μH​

不过，磁场还要引入一个磁矢势A⃗\vec{A}A，使得B⃗=∇×A⃗\vec{B} =\nabla\times\vec{A}B=∇×A. 一般来说，这里还是采取库伦规范条件∇⋅A⃗\nabla\cdot\vec{A}∇⋅A，于是有

∇×B⃗=∇×(∇×A⃗)=∇(∇⋅A⃗)−∇2A⃗=−∇2A⃗=4πcμj⃗\nabla\times\vec{B} = \nabla\times(\nabla\times\vec{A}) = \nabla(\nabla\cdot\vec{A})-\nabla^2\vec{A} = -\nabla^2\vec{A}=\dfrac{4\pi}{c}\mu\vec{j} ∇×B=∇×(∇×A)=∇(∇⋅A)−∇2A=−∇2A=c4π​μj​

即关于磁矢势的泊松方程

∇2A⃗=−4πcμj⃗\nabla^2\vec{A} = -\dfrac{4\pi}{c}\mu\vec{j} ∇2A=−c4π​μj​

一般的，磁矢势表达式为

A⃗=μc∫j⃗(r⃗′)∣r⃗−r⃗′∣dV′\vec{A} = \dfrac{\mu}{c}\int\dfrac{\vec{j}(\vec{r}')}{|\vec{r}-\vec{r}'|}dV' A=cμ​∫∣r−r′∣j​(r′)​dV′

磁偶极子

高斯单位制下，电场公式似乎都比较熟悉，磁场公式则需要小心光速ccc. 所以这里仔细推导一下磁偶极子的定义，我们知道，电偶极子是电场的多级展开的第二项。自然，磁偶极子也是磁场的多级展开的一项。首先，根据

1∣r⃗−r⃗′∣=1r+r⃗⋅r⃗′r3+⋯\dfrac{1}{|\vec{r}-\vec{r}'|} = \dfrac{1}{r} + \dfrac{\vec{r}\cdot\vec{r}'}{r^3} + \cdots ∣r−r′∣1​=r1​+r3r⋅r′​+⋯

将磁矢势的一般表达式展开

A⃗=μc1r∫Vj⃗(r⃗′)dV′+μc1r3r⃗⋅∫Vr⃗′j⃗′dV′+⋯\vec{A} = \dfrac{\mu}{c}\dfrac{1}{r}\int_V\vec{j}(\vec{r}')dV' + \dfrac{\mu}{c}\dfrac{1}{r^3}\vec{r}\cdot\int_V \vec{r}'\vec{j}'dV' + \cdots A=cμ​r1​∫V​j​(r′)dV′+cμ​r31​r⋅∫V​r′j​′dV′+⋯

上式的意义是，设有一个具有电流的导体，其尺度为VVV，求其在很远的r⃗\vec{r}r处产生的磁矢势。而第一项积分体积内没有电流的流入或者流出，所以一定为零。第二项就是我们磁偶极子，首先证明一个式子

∫V(r⃗′j⃗+j⃗r⃗′)dV′=0\int_V (\vec{r}'\vec{j}+\vec{j}\vec{r}')d V' = 0 ∫V​(r′j​+j​r′)dV′=0

这是因为

∇′⋅(j⃗r⃗′r⃗′)=(∇′⋅j⃗)r⃗′r⃗′+j⃗⋅(∇′r⃗′)r⃗′+r⃗′j⃗⋅(∇′r⃗′)=j⃗r⃗′+r⃗′j⃗\begin{aligned} \nabla'\cdot(\vec{j}\vec{r}'\vec{r}') &= (\nabla'\cdot\vec{j})\vec{r}'\vec{r}' + \vec{j}\cdot(\nabla'\vec{r}')\vec{r}' + \vec{r}'\vec{j}\cdot(\nabla'\vec{r}') \\ & = \vec{j}\vec{r}'+\vec{r}'\vec{j} \end{aligned} ∇′⋅(j​r′r′)​=(∇′⋅j​)r′r′+j​⋅(∇′r′)r′+r′j​⋅(∇′r′)=j​r′+r′j​​

体积分可以化为边界上的积分，而边界上的电流为零，所以积分为零。于是磁偶极子项可以写成

A⃗=μc12r3r⃗⋅∫V(r⃗′j⃗−j⃗r⃗′)dV′=−μc12r3r⃗×∫V(r⃗′×j⃗)dV′\vec{A} = \dfrac{\mu}{c}\dfrac{1}{2r^3}\vec{r}\cdot\int_V(\vec{r}'\vec{j}-\vec{j}\vec{r}') dV' = -\dfrac{\mu}{c}\dfrac{1}{2r^3}\vec{r}\times\int_V(\vec{r}'\times\vec{j})dV' A=cμ​2r31​r⋅∫V​(r′j​−j​r′)dV′=−cμ​2r31​r×∫V​(r′×j​)dV′

定义磁偶极子

m⃗=12c∫Vr⃗′×j⃗dV′\vec{m} = \dfrac{1}{2c}\int_V\vec{r}'\times\vec{j}dV' m=2c1​∫V​r′×j​dV′

于是

A⃗=μm⃗×r⃗r3\vec{A} = \mu\dfrac{\vec{m}\times\vec{r}}{r^3} A=μr3m×r​

就和电偶极子的公式比较像了。对于线圈，有m⃗=12c∫r⃗′×j⃗dV′=I2c∮r⃗′×dl⃗′\vec{m} = \dfrac{1}{2c}\int\vec{r}'\times\vec{j}dV'=\dfrac{I}{2c}\oint\vec{r}'\times d\vec{l}'m=2c1​∫r′×j​dV′=2cI​∮r′×dl′，对于任意的平面线圈，有S⃗=12∮r⃗×dl⃗\vec{S} = \dfrac{1}{2}\oint\vec{r}\times d\vec{l}S=21​∮r×dl，所以，有

m⃗=1cIS⃗\vec{m} = \dfrac{1}{c}I\vec{S} m=c1​IS

这样定义的磁偶极子，除了能够消除磁偶极子的磁矢势中的ccc，还使得磁偶极子的在磁场的受力公式更加简洁。

来看看受力，高斯单位制中，洛伦兹力为F⃗=1cv⃗×B⃗\vec{F} = \dfrac{1}{c}\vec{v}\times\vec{B}F=c1​v×B，一般式为

F⃗=1c∫Vj⃗(r⃗)×B⃗(r⃗)dV\vec{F} = \dfrac{1}{c}\int_V\vec{j}(\vec{r})\times\vec{B}(\vec{r})dV F=c1​∫V​j​(r)×B(r)dV

于是，将磁场在原点展开B⃗(r⃗)=B⃗0+r⃗⋅∇B⃗0+⋯\vec{B}(\vec{r}) = \vec{B}_0 + \vec{r}\cdot\nabla\vec{B}_0+\cdotsB(r)=B0​+r⋅∇B0​+⋯，前面已经提到，电流密度的体积分为零，所以展开式第一项积分为零，第二项是磁偶极子受力项

F⃗=1c∫Vj⃗×(r⃗⋅∇B⃗0)dV=1c(∫Vj⃗r⃗dV⋅∇)×B⃗0\vec{F} = \dfrac{1}{c}\int_V\vec{j}\times(\vec{r}\cdot\nabla\vec{B}_0)dV = \dfrac{1}{c}(\int_V\vec{j}\vec{r}dV\cdot \nabla)\times\vec{B}_0 F=c1​∫V​j​×(r⋅∇B0​)dV=c1​(∫V​j​rdV⋅∇)×B0​

前面已经知道∫j⃗r⃗dV=∫(j⃗r⃗−r⃗j⃗)dV\int\vec{j}\vec{r}dV = \int(\vec{j}\vec{r}-\vec{r}\vec{j})dV∫j​rdV=∫(j​r−rj​)dV，而(j⃗r⃗−r⃗j⃗)⋅∇=(r⃗×j⃗)×∇(\vec{j}\vec{r}-\vec{r}\vec{j})\cdot \nabla = (\vec{r}\times\vec{j})\times \nabla(j​r−rj​)⋅∇=(r×j​)×∇，所以

F⃗=(m⃗×∇)×B⃗0=∇(m⃗⋅B⃗0)−m⃗(∇⋅B⃗0)\vec{F} = (\vec{m}\times\nabla)\times\vec{B}_0 = \nabla(\vec{m}\cdot\vec{B}_0)-\vec{m}(\nabla\cdot\vec{B}_0) F=(m×∇)×B0​=∇(m⋅B0​)−m(∇⋅B0​)

后一项为零，所以磁偶极子受力公式为

F⃗=∇(m⃗⋅B⃗)\vec{F} = \nabla(\vec{m}\cdot\vec{B}) F=∇(m⋅B)

这个公式看上去和SI制是一致的，并且与电偶极子的受力公式也是一致的。磁偶极子受到的力矩请自行推导

N⃗=∫Vr⃗×(1cj⃗×B⃗0)dV=m⃗×B⃗\vec{N} = \int_V\vec{r}\times (\dfrac{1}{c}\vec{j}\times\vec{B}_0)dV = \vec{m}\times\vec{B} N=∫V​r×(c1​j​×B0​)dV=m×B

因此，高斯单位制下，磁偶极矩的定义，别忘了分母上还有一个ccc.
