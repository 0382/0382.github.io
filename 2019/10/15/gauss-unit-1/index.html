高斯单位制（其一）

发表于2019-10-15更新于2020-10-30阅读次数

高斯单位制是一个比较古老的单位制，但是在物理学研究中的很多领域还有很大的用处.不过本文不是用来对比高斯单位制和国际单位制的，而是从高斯单位制出发，推导一下电动力学的相关公式.本节主要是电磁学一些基本物理量的定义公式，高斯单位制下和高中学的有较大的区别.

厘米-克-秒制（CGS制）

centimeter-gram-second system以下都简称CGS制。在力学单位上，CGS是一致的，但是在电学单位上有几种变种。本文对此并不感兴趣，只考虑其中之一的高斯单位制。

高斯单位制

电荷的单位

在高斯单位制里，直接定义库伦定律为

F=Q1Q2r2F = \dfrac{Q_1 Q_2}{r^2} F=r2Q1​Q2​​

高斯单位制源于CGS制，因此上式中，力的单位为达因(dyne)，距离的单位为厘米。现在定义电荷的单位为statC，可以写成力学单位的组合

1statC=1g1/2cm3/2s−11 \mathrm{statC} = 1 g^{1/2}cm^{3/2}s^{-1} 1statC=1g1/2cm3/2s−1

静电场

接下来是一些基本电磁学单位的推导。有了电荷，就可以定义电流和电压了。电流的方程还是一样I=ΔQ/ΔtI = \Delta Q/\Delta tI=ΔQ/Δt，单位是 statC/s−1\mathrm{statC/s^{-1}}statC/s−1. 电压（电势）则是根据库伦定律

ϕ=Qr\phi = \dfrac{Q}{r} ϕ=rQ​

电势的单位名称为 statV，1 statV = 1 statC/cm.

接下来是电场强度，库伦定律形式不变

E=Qr2E = \dfrac{Q}{r^2} E=r2Q​

于是电场单位为 statV/cm. 现在就可以搞静电学了。首先是静电场的高斯定律

∮E⃗⋅dS⃗=4π∑Qi=4π∫VρdV.\oint \vec{E} \cdot d\vec{S} = 4\pi\sum Q_i = 4\pi\int_V \rho dV. ∮E⋅dS=4π∑Qi​=4π∫V​ρdV.

这里开始有4π4\pi4π了，请回想一下高斯定理的推导，这个还是很容易的。微分形式就是

∇⋅E⃗=4πρ\nabla \cdot \vec{E} = 4\pi \rho ∇⋅E=4πρ

当然，静电场旋度为零这个关系还是成立的

∇×E⃗=0\nabla\times \vec{E} = 0 ∇×E=0

在CGS制里，这里的电荷密度单位当然就是 statC/cm3\mathrm{statC/cm^3}statC/cm3. 顺带一提，电荷守恒定律为

∂ρ∂t+∇j⃗=0\dfrac{\partial \rho}{\partial t} + \nabla \vec{j} = 0 ∂t∂ρ​+∇j​=0

电流密度矢量的单位，自不必多言。

你也可以试着推导一下电阻，电容之类的，不过本文的目的还是为了确定电磁场公式，这些就不全部给出了。

静磁场

对于静磁场，其出发点是毕奥-萨伐尔定律

dF⃗=kmI1I2dl⃗2×(dl⃗1×r⃗12)r123d\vec{F} = k_m\dfrac{I_1 I_2 d\vec{l}_2 \times (d\vec{l}_1\times \vec{r}_{12})}{r^3_{12}} dF=km​r123​I1​I2​dl2​×(dl1​×r12​)​

不过，前面定义了电荷，电流，长度量纲也是已定义的，所以上式中的常数kmk_mkm​是无法定义为1的。不过，实际上我们知道，在国际单位制中，通过麦克斯韦方程组解出，静电常量kek_eke​和静磁常量kmk_mkm​的关系是

kekm=c2\dfrac{k_e}{k_m} = c^2 km​ke​​=c2

光速是自然界的常数，自然在任何单位制下都是成立的的。因此，kmk_mkm​的值为

km=1c2k_m = \dfrac{1}{c^2} km​=c21​

这么说有点像拿未知的推导已知的，不过我们当然可以先不求出kmk_mkm​的值，直接推导出麦克斯韦方程组，然后再确定kmk_mkm​的值。或者我们也可以直接重新定义kmk_mkm​为式1/c21/c^21/c2，并且把ccc看成一个未知的常数，之后再确定ccc。总之，最终目的是得到没有歧义的高斯单位制下的电磁场方程。不管怎样，高斯单位制下ccc具有速度的量纲。

对于毕奥-萨伐尔定律，可以拆解为两部分：安培力公式

dF⃗=Idl⃗×1cB⃗d\vec{F} = I d\vec{l}\times \dfrac{1}{c} \vec{B} dF=Idl×c1​B

和电流元产生的磁感应强度公式

dB⃗=1cIdl⃗×r⃗r3d\vec{B} = \dfrac{1}{c} \dfrac{I d\vec{l}\times \vec{r}}{r^3} dB=c1​r3Idl×r​

在这个拆解下，磁感应强度具有和电场强度相同的量纲。不过还是另外定义了磁感应强度的单位为 gauss，并且有关系 gauss=statV/cm=cm−1/2g1/2s−1\mathrm{gauss} = \mathrm{statV/cm} = \mathrm{cm^{-1}/2g^{1/2}s^{-1}}gauss=statV/cm=cm−1/2g1/2s−1，但是 1 gauss 并不等于 1 statV/cm，因为磁感应强度显然不能等于电场强度。(所以说还是国际单位制清晰明了。)

虽然历史上 1gauss 的定义最初是在电磁单位制下的，不过，在高斯单位制下，1gauss 可以定义为：电流元磁感应强度公式，电流为III无限长直导线在距离为aaa处产生的磁感应强度大小为

B=2IcaB = \dfrac{2I}{c a} B=ca2I​

如果I=29979245800statC/s,a=1cmI = 29979245800\mathrm{statC/s}, a = 1\mathrm{cm}I=29979245800statC/s,a=1cm，(注意光速应该为CGS单位制，也就是c=29979245800cm/sc = 29979245800\mathrm{cm/s}c=29979245800cm/s)，则有B=2gaussB = 2\mathrm{gauss}B=2gauss，并且有1gauss=10−4T1 \mathrm{gauss} = 10^{-4}T1gauss=10−4T.

现在我们可以做静磁学了，根据电流元磁感应强度公式，导线产生的磁感应强度为

B⃗(r⃗)=1c∫LIdl⃗×(r⃗−r⃗′)∣r⃗−r⃗′∣3\vec{B}(\vec{r}) = \dfrac{1}{c}\int_L \dfrac{I d\vec{l}\times (\vec{r}-\vec{r}')}{|\vec{r}-\vec{r}'|^3} B(r)=c1​∫L​∣r−r′∣3Idl×(r−r′)​

对于具有体积的导体，则应该是

B⃗(r⃗)=1c∫Vj⃗(r⃗′)×(r⃗−r⃗′)∣r⃗−r⃗′∣3dV′\vec{B}(\vec{r}) = \dfrac{1}{c}\int_V\dfrac{\vec{j}(\vec{r}')\times(\vec{r}-\vec{r}')}{|\vec{r}-\vec{r}'|^3}dV' B(r)=c1​∫V​∣r−r′∣3j​(r′)×(r−r′)​dV′

于是磁矢势就是(根据B⃗=∇×A⃗\vec{B} = \nabla \times \vec{A}B=∇×A的关系)

A⃗(r⃗)=1c∫Vj⃗(r⃗′)∣r⃗−r⃗′∣dV′\vec{A}(\vec{r}) = \frac{1}{c}\int_V\dfrac{\vec{j}(\vec{r}')}{|\vec{r}-\vec{r}'|}dV' A(r)=c1​∫V​∣r−r′∣j​(r′)​dV′

对上式作用∇\nabla∇，可以证明

∇⋅A⃗=0∇2A⃗=−4πcj⃗\nabla \cdot \vec{A} = 0 \qquad \nabla^2\vec{A} = -\dfrac{4\pi}{c}\vec{j} ∇⋅A=0∇2A=−c4π​j​

从而可以得到

∇×B⃗=4πcj⃗\nabla \times \vec{B} = \dfrac{4\pi}{c} \vec{j} ∇×B=c4π​j​

由B⃗=∇×A⃗\vec{B} = \nabla \times \vec{A}B=∇×A可得

∇⋅B⃗=0\nabla \cdot \vec{B} = 0 ∇⋅B=0
